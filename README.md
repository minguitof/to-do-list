## How to start the frontend

1. Clone the repository

```bash
git clone https://github.com/minguitof/to-do-list-vue.git
npm install
npm run dev
```

2. Navigate to the frontend folder

```bash
cd Front
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```
